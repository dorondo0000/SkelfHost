# SkelfHost
SelfHost 기능만 스크립트로 사용하기

사용법
1) SimpleWebServer 플러그인을 설치하세요 https://modrinth.com/plugin/simplewebserver/versions
2) 8080 또는 포트를 하나 여세요
3) 서버폴더 바로 안에 리소스팩을 폴더채로 넣고 이름을 pack으로 변경하세요 ( 예시 서버팩/pack/pack.mcmeta )

명령어
/build - 자동으로 폴더를 압축해서 웹서버에 업로드합니다. 니아이피:8080/파일이름.zip 에 들어가면 누구나 파일 다운로드가능
/r - 특정 플레이어 또는 전체에게 리팩 주기
/dr - 드롭박스 링크 추가 또는 삭제

적은 인원의 서버는 이걸로도 충분하지만 사람이 많이몰리면 또는 리팩 용량이 크면 인터넷이 잠시 느려집니다, 그럴땐 드롭박스를쓰세용
